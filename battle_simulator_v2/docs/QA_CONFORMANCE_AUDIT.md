## A. 構造的・共通根因（複数カードに波及）— 優先

### A1. 「全色扱い」「別名（としても扱う）」を発動条件判定で参照していない ★
カードの `canUse`/`dmgBonus` が `stack[0].color` や `name ===` の**生比較**で、`engine._isTreatedAllColors`（全色扱い）や
`card.nameAliases`/`ctx.nameIs`（別名）を見ない。`_isTreatedAllColors` は現状ほぼ特攻判定でしか使われていない。
- 全色扱い系: Q686-689, Q672-673 ほか — hBP01-006 キアラ / hBP01-007 すいせい / hBP02-001 フブキ / hBP03-006 ころね / hBP03-002 ぼたん（`color!=='緑'`）/ hBP03-003 みこ（`==='赤'`）
- 別名(SorAZ=ときのそら/AZKi)系: Q5/Q10/Q13/Q4 — hSD01-006 / hSD01-011 / hSD01-015（しかも `else if` で片方しか発火しない）/ hBP02-078 かなた建設(Q250)
- 別名(ラムダック)系: Q535 — hBP06-096 / hBP06-083（`e.top.name===name` 厳密、重複選択ガードも無し）

### A2. 「対象不在でも推し/SPスキルは使用宣言できる（空振り解決）」原則違反 ☆（実害は小）
公式は「在庫/対象が無くても使用可能、解決時に何もしないだけ」。実装は `canUse` で事前に弾き**使用不可**にしている。
エンジンは `canUse=false` を行動提示しない（engine.js 757/1112）ため仕様差。修正は在庫チェックを `run` 内の「対象無ければreturn」へ寄せる。
- Q296/297 hBP03-001 ルーナ / Q299 hBP03-002 ぼたん / Q300 hBP03-003 みこ / Q309 hBP03-017 ぼたん / Q338 hBP03-082 奏 /
  Q341 hBP03-084 / Q343 hBP03-086 / Q344 hBP03-087 / Q348 hBP03-088 / Q432 hBP05-002 / Q441 hBP05-005 /
  Q237 hBP02-007 カリオペ / Q554 hSD01-020 ホロリスの輪 / Q551 hBP02-084 みっころね24 / Q606 hBP07-092 アーカイブパソコン

### A3. 同時誘発・複数装備の処理順をプレイヤーが選べない ★
複数の「○○した時」が同時誘発した場合や複数装備のダウン処理は、所有/ターンプレイヤーが順序を選ぶ。
実装は固定配列順で逐次解決し、順序選択の決定ポイントが無い。
- Q587 hBP07-004 / Q614 hBP07-028+106 / Q474 hBP03-107等 / Q477-478 / Q259 hSD04-014 / Q257 hSD02-014 / Q649 hBP08-007（ターン終了時効果の解決順）

### A4. デッキ/アーカイブからの登場（putToBack 等）で onEnter・ギフトが誘発しない ★
`ctx.putToBack`（context.js 640）が `onEnter` を呼ばない。手札からの `place` でしか発火しない。
- Q575 hSD13-014 セシリア（hSD13-009/002/001 のデッキ/アーカイブ展開で不発）

### A5. 特殊ダメージ起因のダウンで、発生源装着カードの onOpponentDown が発火しない ★
`context.js dealSpecialDamage` は `_notifySourceDown`（継続効果通知）だけで、装着カードの `triggers.onOpponentDown` を呼ばない。
- Q207 hBP01-115 すいせいのマイク

### A6. 「軽減されない」アーツが被ダメ割り込み(onDamageReceivedReact)を貫通しない ★
`noReduce` を `_applyDamageReceived` の ignoreReduction にしか渡さず、割り込み(V.7 等)経路に伝播していない。
- Q539 hBP01-027 V.7 / hBP06-027 風真いろは（軽減無効アーツに V.7 が効いてしまう）

### A7. サイコロの宣言値/振り直しが rollDie 直叩き経路に反映されない ★
- Q221 hBP01-027 V.7: 「左手に地図」(hSD01-002)の宣言目が V.7 のロールに乗らない（`rollDie` 直呼び、`ctx.rollDice` 非経由）
- Q229 hBP01-123 野うさぎ同盟: 振り直しが1個だけ。ぺこら(hBP01-043)の3個振りで他の確定目が残る

- **UNCERTAIN: 約14件**（要・原文再確認／未実装の可能性）。
