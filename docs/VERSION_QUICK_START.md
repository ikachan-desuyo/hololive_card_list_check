# 🚀 バージョン更新 Quick Start

## すぐに使える方法

### 方法1: 直接実行（推奨）
```bash
node update-version.js 4.11.5 "新機能の説明"
```

### 方法2: npm スクリプト経由
```bash
npm run version-update 4.11.5 "新機能の説明"
```
